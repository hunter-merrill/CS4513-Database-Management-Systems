# CS4513 Database Management Systems
![Java](https://img.shields.io/badge/Java-orange?logo=coffeescript&logoColor=white) ![Transact SQL](https://img.shields.io/badge/Transact_SQL-blue?logo=databricks&logoColor=white) ![LaTeX](https://img.shields.io/badge/LaTeX-008080?logo=LaTeX&logoColor=white)

Project work for CS4513 DBMS. Uses TSQL and JDBC to implement a mock database for maintaining a registry of national park personnel. The code demonstrates key concepts and skills in database design, implementation, and management.

## Skills Demonstrated

- Database schema design
- SQL query writing and optimization
- Java proficiency/JDBC integration
- Familiarity with Azure
